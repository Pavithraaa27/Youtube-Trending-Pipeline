## 🚀 Overview

A **Python-based data engineering pipeline** that fetches trending videos from the YouTube API, processes the data, and stores it in a structured SQLite database.

The system extracts insights and displays the **Top 10 trending videos**, demonstrating a complete **ETL (Extract, Transform, Load) workflow**.

---

## ✨ Key Features

- 📡 Fetch real-time trending videos using YouTube Data API  
- 🗄️ Store structured data in SQLite database  
- 🔄 ETL pipeline (Extract → Transform → Load)  
- 🔐 Secure API key handling using `.env` file  
- 📊 Display Top 10 trending videos  
- 🧹 Clean and modular Python code  

---

## 🏗️ System Architecture

YouTube API → Data Extraction → Processing → SQLite Database → Top 10 Trending Output

## 🛠️ Tech Stack
