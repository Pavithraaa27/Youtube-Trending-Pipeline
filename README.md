YouTube Trending Pipeline

A Python-based data pipeline that collects and stores YouTube trending data using the YouTube API, processes it into a structured database, and displays top 10 trending videos on youtube.

---
FEATURES

- Fetch trending video data using YouTube API
- Store results in SQLite database
- Clean and structured pipeline logic
- Environment-based API key management

---

TECH STACK

- Python 3
- YouTube Data API
- SQLite
- python-dotenv
